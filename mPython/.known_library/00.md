# 📦 Installed Libraries & Their Usage

//////////////////////////////////////////////
### 🔹 **numpy (2.2.6)**

* **What it is**: Core numerical computing library in Python.
* **Why it matters**: Provides **fast arrays, matrices, linear algebra, statistics, and numerical operations**.
* **Usage examples**:

  ```python
  import numpy as np
  arr = np.array([1, 2, 3])
  print(arr * 2)   # [2, 4, 6]
  ```
* **Real world**: Backbone for **machine learning, data science, image processing (with OpenCV)**.

---

//////////////////////////////////////////////
### 🔹 **opencv-python (4.12.0.88)**

* **What it is**: The Python bindings for **OpenCV**, a huge computer vision library written in C++.
* **Why it matters**: Lets you **read, manipulate, and analyze images & videos**.
* **Usage examples**:

  ```python
  import cv2
  img = cv2.imread("test.jpg")
  gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
  cv2.imshow("Gray", gray)
  cv2.waitKey(0)
  ```
* **Real world**: Face detection, object tracking, self-driving cars, AI vision, AR filters.

---

//////////////////////////////////////////////
### 🔹 **opencv-contrib-python (4.12.0.88)**

* **What it is**: Same as `opencv-python` but with **extra (contrib) modules**.
* **Why it matters**: Adds advanced algorithms like **SIFT, SURF, ArUco markers, tracking APIs, 3D reconstruction**.
* **Usage examples**:

  ```python
  sift = cv2.SIFT_create()
  kp, des = sift.detectAndCompute(gray, None)
  ```
* **Real world**: Advanced research, robotics, SLAM, AR marker tracking.

---

//////////////////////////////////////////////
### 🔹 **rich (14.1.0)**

* **What it is**: A library for **beautiful terminal output** (progress bars, syntax highlighting, tables, colors).
* **Why it matters**: Makes CLI tools look **professional and user-friendly**.
* **Usage examples**:

  ```python
  from rich.console import Console
  console = Console()
  console.print("[bold green]Hello, World![/]")
  ```
* **Real world**: Pip-style progress bars, logging dashboards, better developer tools.

---

//////////////////////////////////////////////
### 🔹 **markdown-it-py (4.0.0)**

* **What it is**: A **Markdown parser** for Python (translates Markdown → tokens/HTML).
* **Why it matters**: Used internally by `rich` for rendering rich text/Markdown in the terminal.
* **Usage examples**:

  ```python
  from markdown_it import MarkdownIt
  md = MarkdownIt()
  print(md.render("**Hello** _World_"))
  ```
* **Real world**: Building Markdown editors, documentation generators, terminal apps with Markdown support.

---

//////////////////////////////////////////////
### 🔹 **mdurl (0.1.2)**

* **What it is**: A **URL parsing/normalization helper** used by `markdown-it-py`.
* **Why it matters**: Ensures that links in Markdown are handled safely and correctly.
* **Usage examples**: You won’t usually use it directly; `markdown-it-py` calls it.

---


//////////////////////////////////////////////
### 🔹 **pygments (2.19.2)**

* **What it is**: A **syntax highlighter** for code.
* **Why it matters**: Converts source code into **colorized, formatted text**.
* **Usage examples**:

  ```python
  from pygments import highlight
  from pygments.lexers import PythonLexer
  from pygments.formatters import HtmlFormatter

  code = 'print("Hello, World!")'
  print(highlight(code, PythonLexer(), HtmlFormatter()))
  ```
* **Real world**: Used in Jupyter notebooks, documentation tools, and `rich` for code highlighting.

//////////////////////////////////////////////
### 🔹 **tqdm**

* **What it is**: A Python library for **progress bars** in loops and long-running tasks.

* **Why it matters**: Gives **real-time feedback** about task progress (percentage, ETA, speed), making scripts look **professional and user-friendly**.

* **Usage examples**:

  ```python
  from tqdm import tqdm
  import time

  for i in tqdm(range(100), desc="Processing", ncols=100):
      time.sleep(0.05)  # simulate work
  ```

* **Real world**: Widely used in **data processing, machine learning training loops, file downloads, simulations**, or any script where you want to track progress.
